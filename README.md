# clinic-waiting-display

ふじみ野ひかり眼科の待合室向けデジタルサイネージ用Webアプリケーションです。

## 機能

### 花粉飛散情報
- 月別の花粉カレンダー表示（1月〜12月）
- 対象花粉: ハンノキ属、スギ、ヒノキ、イネ科、ブタクサ属、ヨモギ属、カナムグラ
- 飛散レベル: 多い・やや多い・少ない（色分け表示）
- 今月の花粉情報と対策ポイントを自動表示

### 画面ローテーション
以下の画面が自動で切り替わります：

| 画面 | 表示時間 | 内容 |
|------|---------|------|
| メイン画面 | 30秒 | 花粉カレンダーと対策情報 |
| 花粉注意画面 | 15秒 | 今月注意すべき花粉の警告表示 |
| 対策ポイント | 15秒 | 花粉症対策のアドバイス |
| 画像スライドショー | 可変 | imagesフォルダの画像を順番に表示 |
| ビリヤードロゴ | 30秒 | クリニック名アニメーション |
| 時計画面 | 15秒 | 現在時刻とクリニックロゴ |
| 天気予報 | 30秒 | ふじみ野市の天気（気象庁データ） |
| ロゴアニメーション | 15秒 | パーティクルアニメーション |
| ファミコン風ロゴ | 15秒 | レトロ風アニメーション |

### 対策ポイント表示
- 外出時: マスク・メガネの着用推奨
- 帰宅時: 花粉払い、手洗い・うがい
- 洗濯物: 室内干しで花粉付着防止

## 使用方法

`待合室患者情報提供ディスプレイ.html` をブラウザで開くと、自動的に画面のローテーションが始まります。

## 画像スライドショーの使い方

### 画像の追加方法

1. **画像を準備**
   - ファイル名を `{順番}_{表示秒数}_{説明}.{拡張子}` の形式にする
   - 例: `1_15_花粉症対策.png` → 1番目に表示、15秒間表示
   - 例: `2_10_アイフレイル情報.jpg` → 2番目に表示、10秒間表示

2. **imagesフォルダに画像を入れる**
   ```
   images/
   ├── 1_15_花粉症対策.png
   ├── 2_10_アイフレイル1.png
   ├── 3_20_アイフレイル2.png
   └── ...
   ```

3. **設定ファイルを更新**
   - **Windows**: `update-images.bat` をダブルクリック
   - **Mac/Linux**: ターミナルで `./update-images.sh` を実行

4. **HTMLを開く**
   - 新しい画像がスライドショーに追加されます

### 対応画像形式
- PNG, JPG, JPEG, GIF, WEBP

## 必要な画像ファイル

時計画面で使用するロゴ画像が必要です：
- `ふじみ野ひかり眼科_ロゴ_高画質.png`（カレントディレクトリに配置）

## データソース

- 花粉飛散情報: 環境省 花粉症環境保健マニュアル
- 天気予報: 気象庁

## 技術仕様

- 純粋なHTML/CSS/JavaScriptで構成（フレームワーク不要）
- レスポンシブ対応（`clamp()`によるフォントサイズ調整）
- Noto Sans JPフォント使用
- 各種アニメーション効果（パルス、フロート、パーティクル等）
- 画像スライドショー: ファイル名から順番・表示時間を自動解析

## ファイル構成

```
clinic-waiting-display/
├── 待合室患者情報提供ディスプレイ.html  # メインファイル
├── ふじみ野ひかり眼科_ロゴ_高画質.png    # 時計画面用ロゴ
├── update-images.bat                      # Windows用スクリプト
├── update-images.sh                       # Mac/Linux用スクリプト
├── images/                                # スライドショー画像フォルダ
│   ├── config.json                        # 自動生成される設定ファイル
│   ├── 1_15_花粉症対策.png
│   └── ...
└── README.md
```
