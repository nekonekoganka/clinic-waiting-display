# clinic-waiting-display

ふじみ野ひかり眼科の待合室向けデジタルサイネージ用Webアプリケーションです。

## 機能

### 花粉飛散情報
- 月別の花粉カレンダー表示（1月〜12月）
- 対象花粉: ハンノキ属、スギ、ヒノキ、イネ科、ブタクサ属、ヨモギ属、カナムグラ
- 飛散レベル: 多い・やや多い・少ない（色分け表示）
- 今月の花粉情報と対策ポイントを自動表示

### 画面ローテーション
以下の画面が自動で切り替わります：

| 画面 | 表示時間 | 内容 |
|------|---------|------|
| メイン画面 | 30秒 | 花粉カレンダーと対策情報 |
| 花粉注意画面 | 15秒 | 今月注意すべき花粉の警告表示 |
| 対策ポイント | 15秒 | 花粉症対策のアドバイス |
| 時計画面 | 15秒 | 現在時刻とクリニックロゴ |
| 天気予報 | 30秒 | ふじみ野市の天気（気象庁データ） |
| アイフレイル情報 | 10〜30秒 | 目の健康に関する啓発情報 |
| ロゴアニメーション | 15〜30秒 | クリニックロゴのアニメーション表示 |

### 対策ポイント表示
- 外出時: マスク・メガネの着用推奨
- 帰宅時: 花粉払い、手洗い・うがい
- 洗濯物: 室内干しで花粉付着防止

## 使用方法

`待合室患者情報提供ディスプレイ.html` をブラウザで開くと、自動的に画面のローテーションが始まります。

## 必要な画像ファイル

以下の画像ファイルが同じディレクトリに必要です：

- `花粉症にはメガネとマスクが有効です.png`
- `ふじみ野ひかり眼科_ロゴ_高画質.png`
- `アイフレイル1.png`
- `アイフレイル2.png`
- `アイフレイル3.png`
- `アイフレイル4.png`

## データソース

- 花粉飛散情報: 環境省 花粉症環境保健マニュアル
- 天気予報: 気象庁

## 技術仕様

- 純粋なHTML/CSS/JavaScriptで構成（フレームワーク不要）
- レスポンシブ対応（`clamp()`によるフォントサイズ調整）
- Noto Sans JPフォント使用
- 各種アニメーション効果（パルス、フロート、パーティクル等）
